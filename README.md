## Configuración Inicial

Este proyecto depende de [AbletonJS](https://github.com/leolabs/ableton-js). Para configurarlo:

1. Descarga la [última versión del script del controlador](https://github.com/leolabs/ableton-js/releases/latest)
2. Renombra la carpeta `midi-script` a `AbletonJS`
3. Copia la carpeta en: `C:\ProgramData\Ableton\Live 12 Lite\Resources\MIDI Remote Scripts`
4. En Ableton Live, configura `AbletonJS` como controlador MIDI remoto
